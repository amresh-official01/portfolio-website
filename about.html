My Story

Hello! I'm Amresh Kumar, a passionate Web Developer currently pursuing a B.Sc. in Information Technology. My journey into programming started in 2023 when I began exploring how websites and digital systems work. What started as curiosity quickly turned into a strong interest in building modern web applications and learning new technologies.

During my learning journey, I developed several practical projects, including a College Management System and an E-Commerce Shopping Website. The College Management System was designed to manage students, faculty, attendance, exams, and fee payments through a single digital platform, giving me hands-on experience in full-stack web development.

As a developer, I focus on building responsive, user-friendly, and efficient web applications using technologies like HTML, CSS, JavaScript, PHP, and MySQL. I enjoy solving real-world problems through code and continuously improving my skills to create better digital solutions.
